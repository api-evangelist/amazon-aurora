# Amazon Aurora

Amazon Aurora is a MySQL and PostgreSQL-compatible relational database built for the cloud that combines the performance and availability of traditional enterprise databases with the simplicity and cost-effectiveness of open source databases.

## APIs

- **Amazon Aurora API** - The Amazon Aurora API is accessed through the Amazon RDS API and enables programmatic management of Aurora DB clusters, instances, snapshots, parameter groups, and other database resources for both MySQL and PostgreSQL compatible editions.
